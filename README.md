## 🔒 Telegram Fork (Private Development)

Мой главный и самый амбициозный проект — **независимый форк Telegram** на базе официальной библиотеки **TDLib**.

Это альтернативная экосистема для тех, кому нужна 100% автономия. Проект создается так, чтобы мессенджер работал на ваших **собственных приватных серверах**, полностью исключая сторонний контроль над переписками и логами.

* 🚦 **Текущий статус**: Закрытая разработка (Private Alpha).
* ⚙️ **Архитектура**: Базируется на мощном и безопасном движке **TDLib**.
* 👥 **Как принять участие?** Доступ к ранним сборкам предоставляется индивидуально. Нажмите на бейдж ниже, чтобы написать мне в ЛС, или создайте Issue с тегом `[Beta Request]`.

## 🚀 Как поднять свой мессенджер? (Концепция для форкеров)

Мы проектируем проект так, чтобы **любой желающий мог форкнуть этот репозиторий** и развернуть собственную сеть мессенджера за несколько минут. 

**Что потребуется от вас:**
1. Нажать кнопку **Fork** на этом репозитории.
2. Иметь собственную **мощную серверную часть** (VPS/VDS с хорошим процессором и SSD для обработки базы данных TDLib и трафика).
3. Запустить готовый Docker-компоуз (инструкция и скрипты будут добавлены в публичный релиз). Серверная часть автоматически развернет изолированную среду для вашей независимой копии мессенджера.
