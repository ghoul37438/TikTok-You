## ⚠️ Дисклеймер (Важно перед установкой)

1. **Я НЕ являюсь разработчиком этого мода.** Этот репозиторий — публичное зеркало для вашего удобства. Файлы просто выгружаются сюда из закрытых источников.
2. Проект создан для того, чтобы люди перестали сидеть на кривых сборках (вроде *TikTok Cloud*, *TikTok Prime*, *TikTok Central* и им подобных), которые стабильно жрут говно и радуют вас аварийными вылетами каждые одну-две недели.
3. Здесь публикуются **только официальные стабильные релизы и рабочие бета-версии**.
4. **Важное требование по архитектуре:** Поддержка старых процессоров ARM-7 полностью прекращена. Все сборки в этом репозитории предназначены **строго для новой архитектуры ARM-8(иногда могу выкладывать старую сборку **НО** вней нет новых функций(как была прекразена поддержка ARM-7))**.

> 🚨 **ОБЯЗАТЕЛЬНО:** Перед установкой любой сборки отсюда полностью удалите оригинальный TikTok или любой другой кривой мод, который стоял у вас до этого! Иначе приложение просто не установится или будет конфликтовать.
