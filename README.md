## 📌 Навигация по репозиторию (Repository Branches)

Проект разбит на изолированные ветки и директории. Выбирайте нужное направление:

* **🔒 [Telegram Fork (DigGram)](#-telegram-fork-diggram-private-development)** — Мой главный проект. Автономный мессенджер для развертывания на личных мощностях в обход официальной централизованной экосистемы.
* **🌐 [VPN Hub](./VPN/)** — Постоянно обновляемый архив рабочих прокси-протоколов, слитых ключей (`.vpn`/`.txt`) и мега-паков для обхода любых блокировок.
* **🤖 [Extra Plugins](./extra-plugins/)** — Скрипты автоматизации, юзерботы, утилиты для очистки чатов и модули модификации интерфейса.
* **📱 [TikTok-You](./TikTok-You/)** — Чистые официальные релизы и беты мода под архитектуру ARM-8 без рекламы, багов и аварийных вылетов.

## 🔒 Telegram Fork (DigGram) · Private Development

Альтернативная серверная и клиентская архитектура, использующая мощный и безопасный движок **TDLib** (Telegram Database Library). Любой желающий сможет форкнуть проект и поднять суверенную сеть мессенджера на своей инфраструктуре.

* **🚦 Текущий статус**: Закрытая разработка (Private Alpha). Добавление новых функций
* **⚙️ Архитектура**: Полная инкапсуляция баз данных и трафика внутри вашего изолированного контейнера.
* **👥 Как принять участие?** Доступ к ранним сборкам предоставляется индивидуально. Нажмите на бейдж ниже и отправьте заявку в личные сообщения:
